## 이미지 다운로드 방법

1. 현재 사용 중인 이미지 URL에서 이미지를 다운로드:
   - 브라우저에서 이미지 URL 열기
   - 우클릭 → "다른 이름으로 이미지 저장"
   
2. 파일명을 위 목록대로 변경

3. 이 `images/` 폴더에 저장

## 권장 이미지 사양

- **갤러리 이미지**: 1360x680px (2:1 비율)
- **카드 이미지**: 640x360px (16:9 비율)
- **로고**: PNG (투명 배경)
- **행성**: PNG (투명 배경)
- **포맷**: JPG (사진), PNG (로고/아이콘)
- **파일 크기**: 각 이미지 500KB 이하 권장
